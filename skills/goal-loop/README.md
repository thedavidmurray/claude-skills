# Goal Loop

Bounded autonomous execution skill for Claude Code. Plan, act, test, review, iterate.

## Install

```bash
/plugin install goal-loop@claude-skills
```

## Usage

Ask Claude to "run a goal loop on this" or "keep iterating until done".

## What it does

- Defines acceptance criteria up front
- Executes in max 5 cycles
- Reports progress each cycle
- Stops on completion or when blocked
